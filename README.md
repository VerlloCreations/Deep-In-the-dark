# Deep-In-the-dark
It is a platformer minigame of a child traveling in a forest looking for coins and gems to buy in the store and unlock cards. It is my first video game so it has no story and nothing else, there are just many challenges that I added to test my skills while at the same time submitting it as a project for the university.

In this repository I only uploaded the scripts of my game.
The audio and sprites are not mine; they are from a course and the Unity store. Since it is just a practice project, I didn’t create almost any sprites except for a few, but they are not of great importance.
